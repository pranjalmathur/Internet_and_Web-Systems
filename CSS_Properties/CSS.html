CSS Properties

margin:

To create space around an element. For example: y px space from the parent body, etc. It has 4 sub components: top, bottom, right and left (specified as: margin-top: 10px) Syntax: margin: 0px 1px 2px 3px (top, right, bottom, left)

padding:

To create space around the element within the boundary of that element. It creates an empty space around it. It has 4 sub components: top, bottom, right and left (specified as: padding-top: 10px) Syntax: padding: 0px 1px 2px 3px (top, right, bottom, left)

text-decoration:

To add decoration to the specified text. Eg: underline, overline, etc. It is usually followed by text-decoration-style which provides attributes to the property.

position:

To specify the position of an element in the document.

Static: The element is positioned according to the normal flow of the document. Other styling properties have no effect on it.

Relative: The element is positioned relative to normal flow. Properties like left, right, etc. is used to give the relative position to the element.

Absolute: Used to place the element relative to the immediate ancestor. Properties like top, bottom, etc. are used to place the element relative to the parent element.

Fixed: Fixed element is removed from the normal document flow. It is positioned relative to the screen's viewport and doesn't move when scrolled.

overflow:

To provide details about the overflowing content from the element. Can be used to add x and y scroll bars to the content, so as to avoid the overflowing content. Content can be hidden if required or made visible.

border:

Used to specify the border property like solid, etc. Border-style is used to set the type of border Border-width is used to thickness of the border.

box-sizing:

This property is used to specify sizing properties (width and height) that it should include. Whether to include border, padding and margin in the given size, etc. Border-box: includes everything except margin.

float:

To specify the float property either left or right. To move the content relative to the parent element.

z-index:

To maintain the overlapping of the elements on the webpage. It is usually done relative to the parent element. Higher index number has the priority to overlap the other element.

display:

To specify how the element box will be displayed on the page. Default is inline where the elements are placed in a line. It can be use to give them a block structure (usually a new line)

transition:

To specify how an element will change its properties. Example: going from width: 10 to width: 100, transition property will tell how fast/slow this transition would be.

box-shadow:

To provide background shadow to the element box. We can specify the boldness, width of shadow lining, type like blur and color.

border-radius:

To provide rounded edges to the element box. Single value gives the specified circular edges to all 4 corners, while it can be specified for each corner of the element. Four values - border-radius: 1px 0px 0px 1px:

resize:

To specify whether the element could be resized by the user. One can restrict by giving none as the attribute. Vertical, horizontal, both are other few options.

Others:

Properties like: color (for text), font (for text), background-color (to set color of the background), height(in pixel or percentage), width(to set the width of an element), etc. are quite mainstream and are used more or less with every element.
